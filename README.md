# TimesTable 
It's a simple TimeTable project.
There use :
 + ProgressBar(Horizontal)
 + ListView
 
 In This project you learn -
  1. ArrayAdapter
  2. ArrayList
  3. How to work with ProgressBar
  4. SeekBar
  5. Work with Handler class
  6. Also work with setOnSeekBarChangeListener
